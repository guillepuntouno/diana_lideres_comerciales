# 💻 Web - Aplicación Web

Esta carpeta contiene código específico para la versión web de DIANA.

## Estructura:

- **vistas/**: Pantallas específicas para web
  - dashboard/
  - reportes/
  - datos_maestros/
  - configuracion_sistema/
  
- **widgets/**: Widgets específicos para web
  - Tablas de datos avanzadas
  - Gráficos y visualizaciones
  - Componentes de administración

## Características web:
- Interfaces optimizadas para pantallas grandes
- Manejo de datos maestros
- Reportería y analytics
- Configuración del sistema
- Administración de usuarios
